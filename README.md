# NinjaPortal

Decentralized portal with IPFS integration, zero-knowledge proof circuits (circom/snarkjs), multi-language support, and interactive maps. Part of the ninja-craft ecosystem.

## Stack

- **Frontend:** React + Vite + TailwindCSS + Three.js + Leaflet + react-simple-maps
- **Backend:** Express + PostgreSQL + Drizzle ORM
- **Web3:** ethers.js, wagmi, Solidity contracts
- **ZK:** circom circuits + snarkjs
- **IPFS:** ipfs-http-client for decentralized storage
- **i18n:** react-intl + Google Cloud Translate

## Setup

```bash
npm install
npm run dev
```

See `IPFS_FEATURES.md` for IPFS documentation.
